The basics - What is an operating system?

According to Wikipedia, an operating system is software that manages hardware, software, resources, and provides some amount of services for program execution.

Both unix style operating systems and Windows both give users the ability to interface with the computer via command line or GUI. The most common set of Unix operating systems belong to one distrubution of Linux.

Differences between Windows and Unix operating systems

Windows was designed around the GUI, and some more advanced tasks can be performed using the command line. Linux can be used the same way, or be run entirely from command line depending entirely on the Linux distrubution chosen.

A Few Advantages of Windows

  • Easier to learn
  • Better backward compatibility

A Few Advantages of Linux

  • Better stability
  • Better processing power
  • Better built-in security

Summary

The operating system that you choose should be based off of your needs. If you need something to just simply use that works well with little effort, then Windows may suit your needs. Unbuntu is a linux distrubution that is known for being very easy to setup and use. If you need something that requires a more custom setup, that needs to be fast, lightweight, and roubust, then a form of Unix may be the right choice for you.
